一. 按源中的id抓取

uri: /simpleCrawler

参数：

crawlerName: shuqiById、mianFeiTXT

data: 对象类型 属性有：id

示例：

{
"crawlerName": "shuqiById",
  "data":{
	"id":7031591
  }
}


二. 数据更新

uri: /simpleCrawler

参数：

crawlerName: shuqiUpdater、mianFeiTXTUpdater

data: 对象类型 可选属性：bookId，我库中的书id。不提供则为全库更新

示例：

{
"crawlerName": "mianFeiTXTUpdater",
  "data":{
	"bookId":2010194
  }
}


三. 搜索

uri: /search

参数：

name: 搜索输入（书名、作者名等）

andCrawl: 是否搜索完就抓取，默认False

top: 取搜索结果的前几个，默认5

crawlerName: 指定数据源，默认MianfeiTXT

示例：

{
    "name": "大主宰",
    "andCrawl":true,
    "top":2,
    "crawlerName":"mianFeiTXT"
}
